# 화려한 이력서 프로젝트

Vite + React로 만든 다크테마 기반의 화려한 이력서 웹사이트입니다.

## 특징

- 🌙 다크테마 디자인
- 🎴 카드 기반 레이아웃
- ✨ Framer Motion을 활용한 부드러운 애니메이션
- 📱 반응형 디자인
- 🎨 그라데이션과 글래스모피즘 효과

## 설치 및 실행

### 1. Node.js 설치
먼저 Node.js가 설치되어 있어야 합니다. [Node.js 공식 사이트](https://nodejs.org/)에서 다운로드하세요.

### 2. 의존성 설치
```bash
npm install
```

### 3. 개발 서버 실행
```bash
npm run dev
```

브라우저에서 `http://localhost:5173`으로 접속하세요.

### 4. 빌드
```bash
npm run build
```

## 커스터마이징

`src/components/Resume.jsx` 파일을 수정하여 본인의 정보로 변경하세요:

- 이름, 직책, 연락처
- 자기소개
- 기술 스택
- 경력 사항
- 프로젝트
- 학력

## 기술 스택

- React 18
- Vite
- Framer Motion
- CSS3
